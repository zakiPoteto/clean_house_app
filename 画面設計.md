# 掃除記録・習慣化アプリ

画面遷移図（Screen Flow）

---

## 1. 画面一覧（MVP + 拡張）

### MVP 画面

- ホーム画面（タスク一覧）
- タスク追加画面
- タスク詳細画面
- タスク編集画面
- CSV インポート / エクスポート画面

### 拡張画面

- フィルタ・並び替え画面
- 通知設定画面
- チケット管理画面
- 初回オンボーディング画面

---

## 2. 画面遷移図（Mermaid 記法）

> ※ GitHub / Obsidian / Mermaid 対応 Markdown でそのまま描画可能

```mermaid
flowchart TD
    A[初回オンボーディング画面] -->|初回のみ| B[ホーム画面<br/>タスク一覧]

    B -->|＋ボタン| C[タスク追加画面]
    C -->|保存| B
    C -->|キャンセル| B

    B -->|タスク選択| D[タスク詳細画面]

    D -->|編集| E[タスク編集画面]
    E -->|保存| D
    E -->|キャンセル| D

    D -->|今日掃除した| B
    D -->|削除| B

    D -->|カレンダー追加| F[外部カレンダーアプリ]

    B -->|フィルタ/並び替え| G[フィルタ・並び替え画面]
    G -->|適用| B
    G -->|リセット| B

    B -->|設定/バックアップ| H[CSVインポート/エクスポート画面]
    H -->|完了| B

    B -->|通知設定| I[通知設定画面]
    I -->|保存| B

    D -->|延期/スキップ| J[チケット管理画面]
    J -->|チケット使用| B
```
